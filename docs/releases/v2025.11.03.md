# HoundMind v2025.11.03 — Guided installer launcher, vendor‑aligned install docs, and telemetry version sync

> Author: 7Lynx · Doc Version: 2025.11.03

Release tag: `v2025.11.03`

## Highlights

- Guided installer (`scripts/pidog_install.py`) now includes launch options:
  - Start CanineCore (main or interactive control menu)
  - Start PackMind with an optional `PACKMIND_CONFIG` preset
  - Full menu covers: vendor installs (Robot HAT 2.5.x, Vilib, PiDog), I2S audio setup, vendor demo, import/I2C checks, HoundMind deps for Pi 4/5 (full) or Pi 3B (lite), and launching systems.
- Raspberry Pi install guide (`docs/INSTALL.md`) is fully aligned with SunFounder’s official steps and reorganized:
  - A0 (OS imaging) → A1 (vendor modules) → A2 (checks/calibration) → A3 (deps by model) → A4 (run HoundMind)
  - Clear separation between CanineCore and PackMind, with a note that the guided installer can launch either.
  - Pi 3B guidance: 32‑bit OS recommended; 64‑bit supported with lite deps.
- Telemetry server: version displayed now comes from `packmind.__version__` to prevent drift.

## Upgrades

- `packmind.__version__` bumped to `2025.11.03`.
- README/doc headers updated to 2025.11.03 where applicable.

## Developer notes

- Some file/module headers still show 2025.11.01/02; those will update on functional changes.
- Optional tools for setup verification:
  - `tools/list_audio_devices.py` — enumerate audio devices
  - `tools/camera_check.py` — list/probe cameras and save a frame

## How to run (Pi)

```bash
python3 scripts/pidog_install.py
# Use options 8/9 to launch CanineCore, 10 for PackMind (with optional preset)
```
